本次实现概要
- 目标：提供一个简单、可用的网页计算器，支持基本四则运算、键盘输入、清除等功能。
- 产出：一个独立的小网页应用，结构清晰，便于后续扩展。
- 思路与实现点：
  - 前端使用纯 HTML/CSS/JavaScript，方便快速迭代和部署。
  - 运算逻辑采用直接求值（通过 JavaScript 的 Function 防护性使用，适合本地演示环境）。
  - 支持键盘输入：数字、点、小数点、运算符、Enter/=、Backspace、Escape、括号（可扩展）。
  - UI 使用网格布局，按钮布局直观，AC、DEL、÷、×、+、-、=、0-9、. 等常用按钮齐全。

新建的文件（Evolver 任务产出）
- web-calculator/index.html
  - 结构化的界面入口
  - 按钮布局：AC、DEL、÷、×、7、8、9、+、4、5、6、-、1、2、3、=、0、. 等
- web-calculator/style.css
  - 视觉风格、响应式适配、按钮交互效果
  - 4 列网格布局、span2 用于 0/AC 等跨两列
- web-calculator/script.js
  - 逻辑实现：
    - 维护表达式字符串 expression
    - 点击按钮组装表达式、清空、删除、计算
    - 计算使用 sanitized 表达式，将 ×/÷ 映射为 * /
    - 通过 Function 执行表达式并显示结果
    - 键盘事件支持：数字、.、+、-、*、/、Enter、Backspace、Escape、(、) 等
  - 错误处理：计算异常显示 Error，并清空表达式

如何本地测试
- 直接在浏览器打开：web-calculator/index.html
- 或用简单静态服务器（如 Node/Python/各种 http-server 工具）：
  - 例如：在项目根目录执行
    - npx http-server web-calculator -p 8080
  - 然后访问 http://localhost:8080
