如何本地运行
- 直接打开 web-calculator/index.html 即可使用（无需任何后端）
- 也可以用一个简单的静态服务器，本地浏览：
  - 进入目录：cd web-calculator
  - 启动静态服务器（以 Python 为例）：
    - Python 3: python3 -m http.server 8000
    - 打开浏览器访问 http://localhost:8000
- 使用提示
  - 鼠标点击数字、点号和运算符来构造表达式
  - 使用键盘：0-9、.、+、-、*、/、%、(、) 直接输入；回车或等号 Enter 计算；Backspace 删除；Escape 清屏
