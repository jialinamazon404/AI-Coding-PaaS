Simple two-operand calculator

Overview
- Supports: addition (+), subtraction (-), multiplication (*), division (/)
- CLI: python3 ops/calculator/cli.py <a> <op> <b>

Examples
- python3 ops/calculator/cli.py 3 + 4
- python3 ops/calculator/cli.py 10 / 2
- python3 ops/calculator/cli.py 6 * 7

Notes
- The calculator is designed for straightforward two-operand expressions. For more complex expressions, consider extending with a proper parser or evaluating expressions safely.
- You can import and reuse the functions from calculator.py in other Python modules if needed.
